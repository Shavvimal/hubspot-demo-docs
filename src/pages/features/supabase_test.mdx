### Detailed Instructions

1. **Access Supabase Dashboard**
   - Go to [Supabase Dashboard](https://app.supabase.com)
   - Sign in with your account credentials
   - Select your organization from the dropdown menu

2. **Navigate to Project Settings**
   - Select your project from the projects list
   - Ensure you have Owner or Admin permissions

3. **Access User Management**
   - Click on "Authentication" in the left sidebar
   - Select "Users" from the submenu
   - You'll see a list of all current users

4. **Invite New User**
   - Click the "Invite User" button in the top right
   - Enter the user's email address
   - Choose the appropriate role:
     - **Owner**: Full access to all project settings
     - **Developer**: Can manage data but not billing
     - **Read-only**: Can view but not modify data

5. **Configure Permissions (Optional)**
   - Set custom database permissions
   - Configure service role access
   - Define Row Level Security (RLS) policies

6. **Send Invitation**
   - Review all settings
   - Click "Send Invitation"
   - The user will receive an email with signup instructions
